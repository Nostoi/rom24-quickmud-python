# Archive Directory

This directory contains historical documentation, session reports, and development artifacts.

## Structure

- **agents/** - Agent configuration and execution documentation
- **sessions/** - Session summaries and autonomous run reports
- **reports/** - Completion reports and status updates
- **planning/** - Planning documents and implementation roadmaps
- **parity/** - ROM parity tracking and analysis
- **specs/** - Technical specifications and analysis reports

## Purpose

These files document the development history and decision-making process but are not required for understanding or using QuickMUD. They are preserved for reference and historical context.
